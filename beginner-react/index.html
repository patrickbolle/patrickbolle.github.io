Free Tutorials For Beginners Learning React

Category: React     Tags: Javascript, React, Tutorial,

After learning Meteor with Meteor’s custom templating library, Blaze, I sought out to learn React - the internet’s newest hype project.
React is Facebook’s child, acting as the V within the MVC context. From my experience so far, writing applications with the help of React has been incredibly fast and easy once you get over the scary starting phase.

Now that I’ve been developing with React for a couple of months, I thought I would highlight some of the resources I found to help me understand how/why React works the way that it does.

PS. I will be adding to this post as I discover more guides, send me an email if you have any good resources to share.

Free Tutorials/Guides

  • First and foremost, take a good look at Facebook’s own documentation on React. It isn’t the most indepth documentation, but it will get you started in thinking the “React way” - https://facebook.github.io/react/docs/getting-started.html
  • Second take a look at React JS For Stupid People by Andrew Ray. It’s a fantastic introduction into React concepts for anybody familiar with programming logic.
  • Assuming you know something (even a little bit) about Node and NPM, this Nodeschool.io command-line workshop is fantastic to get your head around the basics of React (components). https://github.com/kohei-takata/learnyoureact
  • LevelUpTuts is a fantastic Youtube channel for learning programming
    • This ‘React For Everyone’ Tutorial series if absolutely fantastic and in my opinion is essential to learn React
    • and.. If you are a Meteor developer, LevelUp’s Meteor + React series is also incredible
  • Egghead.io has a great React note-taking application tutorial, only catch is you have to enter your email address to view the tutorials. Nevertheless, egghead is a fantastic online programming site, lots to learn on there.

General

  • If you use Atom, Github’s open source text editor, the React plugin is a fantastic tool.
    • For example, typing the command “rcc6” will create a full React ES6 component, extremely useful for writing quick React apps.


That’s all for now, but I will be adding to this list as I discover other resources. Good luck!
