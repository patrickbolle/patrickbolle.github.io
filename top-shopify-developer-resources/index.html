Top Resources for Shopify Developers (2017)


Patrick Bollenbach

Jan 29th 2017

While working with the Shopify platform, I’ve found a variety of resources for developers and Shopify Partners. Shopify doesn’t have a great compilation article of this useful material, so I figured I could compile all of my favourite snippets, libraries, and articles that I’ve used over the past year.

When I started building client sites with Shopify I would have loved a list of resources like this, so I hope somebody stumbles upon this and finds it helpful.

If you have a project/post/snippet that you want me to share here, send me an email and I’ll add it to the list.

Shopify Code Snippets

Freak Designs’ Code Snippets - https://github.com/freakdesign/Shopify-code-snippets
Joelvh’s Shopify Hacks - https://github.com/joelvh/shopify_hacks
Gocomet’s Snippets - https://github.com/gocomet/snippets
ZakHardage’s Shopify Wholesale Code - https://github.com/zakhardage/Shopify-Wholesale

Useful Shopify Libraries/Frameworks

Christopher Gregory’s Shopify Node API (my favourite Node-Shopify integration library) - https://github.com/christophergregory/shopify-node-api
Shoppad’s UptownCSS, CSS library that helps your app look very similar to the Shopify interface - http://www.uptowncss.com/
Shopify’s Timber CSS Framework/Template, perfect for starting a custom theme - https://github.com/Shopify/Timber
Shopify’s Buy SDK Javascript Library - https://github.com/Shopify/js-buy-sdk
GLD110’s Embedded App Frontend Framework, great for making your embedded apps fit in to the Shopify look and feel - https://github.com/GLD110/Shopify-Embedded-App-Frontend-Framework
Shopify’s Themekit, for building themes on your local computer instead of within the Shopify “Edit HTML/CSS” editor - https://github.com/Shopify/themekit
Discolab’s Cart.js - great library to help make the sometimes complicated AJAX cart easier - https://github.com/discolabs/cartjs

Shopify Partner Resources

Freak Designs’ ShopifyFD - https://github.com/freakdesign/shopifyFD
Forsbergplustwo’s Shopify Partner Metrics (this is really cool) - https://github.com/forsbergplustwo/partner-metrics-for-shopify
50 Best Shopify Resources Blog Post - http://bootstrappingecommerce.com/50-of-the-best-shopify-resources/

That’s all for now, will keep adding to this list when I find more links.
