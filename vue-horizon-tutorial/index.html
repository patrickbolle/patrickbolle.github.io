Vue JS + Horizon - Full Stack Tutorial (Part 1)


Aug 29th 2016

Introduction

I’m really into Vue JS right now. It’s a simplified approach to the view layer frameworks that React is currently conquering. It’s incredibly easy to learn and very powerful for when your app gets complex, it’s essentially everything I want in a framework (and the docs are amazing!). I started out connecting it to an Express API with RethinkDB on the backend. It worked out well but it’s sort tough to get websockets working properly all the time.
Thankfully the RethinkDB team released Horizon.io, it’s sort of like their answer to Meteor - but much much less opinionated allowing us to use any front end framework we choose. It’s also got web sockets built in to interact with RethinkDB’s changefeeds, meaning 100% real time data with no extra effort.
In their own words :

RethinkDB Horizon is an open-source developer platform for building realtime, scalable web apps. It is built on top of RethinkDB, and allows app developers to get started with building modern, engaging apps without writing any backend code.

I started my Javascript journey with Meteor which was a great intro to the JS world, but very limiting after a while (thanks to Mongo). Meteor has been going through some big (controversial) changes recently with them ditching their built in framework Blaze, and removing their free Meteor hosting service (which they promised would be free forever…) - so the Meteor community is sort of in shambles at the moment.
Nevertheless, I sought out another tech stack for my web apps and once I stumbled upon Vue and Horizon it immediately became my Meteor replacement. Fast prototyping + development, real time data, and a growing community.

What We Will Build

This is part 1 of 3 tutorials on building a full stack note-taking app with Vue JS and Horizon.
It is a simple application but it will show you the power of the two frameworks and will be a great starting point for future projects.
The tutorials will go over the following topics -

  • Installation
  • Connecting the two frameworks
  • CRUD Operations
  • Accounts + Authentication with Auth0
  • Deployment to Digital Ocean

So let’s get started!
